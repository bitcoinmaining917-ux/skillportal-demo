# Design Brief: Indian Government Portal

**Aesthetic:** Classic formal government portal. Official, authoritative, clean.

**Palette:**
| Token | OKLCH | Usage |
| --- | --- | --- |
| Primary (Deep Blue) | 0.32 0.08 270 | Header, buttons, links |
| Secondary (Teal) | 0.56 0.15 200 | Navigation, accents, active states |
| Neutral (White) | 0.99 0 0 | Background, cards |
| Muted | 0.94 0.01 270 | Subtle borders, disabled states |

**Typography:** DM Sans (body), Fraunces (display). Formal hierarchy via weight and size.

**Zones:**
| Zone | Treatment |
| --- | --- |
| Header | Deep blue (0.32 0.08 270) with white text, solid bar |
| Navigation | Light teal (0.56 0.15 200), white text, divider below |
| Content | White background (0.99 0 0) with blue borders on cards |
| Footer | Dark blue matching header, white text, border-top |
| Borders | Light grey (0.93 0.01 270) on cards and tables |

**Layout:** Grid-based card layout. Table-style data sections. Minimal spacing, formal alignment.

**Motion:** Subtle transitions (0.3s), no animations. Formal, restrained interaction.

**Constraints:** No authentication UI, no search filters, no PDF exports, no admin panel. Pure display portal.
